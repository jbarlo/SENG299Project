Rules
Rule 1:
black tokens start the game.
white is compensated with 6.5 points for going second.
Rule 2:
once a token is placed at an intersection, it may not be moved.
 
Rule 3:
a token is removed when is has no liberties remaining.
 
Rule 4:
a player may not make a move that returns the board to the
previous state (ko rule).
Rule 5:
a player may not place a token where it would have no liberties,
unless is deprives an enemy token of its last liberty.
Definition:
a liberty is a vancant space adjacent to a token.
 
Definition:
an army is a collection of same coloured tokens adjacent to one another.
 
Buttons
Pass:
skips the player's turn without placing a token.
 
Board Colour:
changes the background colour of the board.
 
Team A Colour:
changes the colour of Team A's tokens, which defaults to black.
 
Team B Colour:
changes the colour of Team B's tokens, which defaults to white.
 
Board Size:
re-adjusts the size of the number of intersections on the board.
NOTE: this returns the game state to an empty board.
Game Mode:
changes the opponent of the game
NOTE: this returns the game state to an empty board.
Replay:
reviews the previous moves, with 3 second intervals
NOTE: this returns the game state to an empty board.
